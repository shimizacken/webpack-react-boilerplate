Simple `React` `webpack` web app

<h2>
  usage
</h2>

clone to local folder:
```bash
git clone https://github.com/shimizacken/webpack-react-boilerplate.git
```

install:
```bash
npm install
```

run the webapp:
```bash
npm start
```

build:
```bash
npm run build-dev
```

build for production:
```bash
npm run build-prod
```
